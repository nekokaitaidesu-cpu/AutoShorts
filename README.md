# AutoShorts

YouTube Shorts を自動的に次の動画へ進める Android アプリ。
縦画面・横画面の両方に対応。

---

## セットアップ手順

### 1. Android Studio でビルド
1. Android Studio でこのフォルダを開く
2. `Build > Generate App Bundles or APKs > Generate APKs` をクリック
3. `app-debug.apk` が生成される（`app/build/outputs/apk/debug/`）

### 2. APK のインストール
```
adb install app/build/outputs/apk/debug/app-debug.apk
```
または、生成した APK ファイルを端末に転送して直接インストール。

> **Android 13 以降でアクセシビリティが「制限付き設定」になる場合**
> 設定 → アプリ → AutoShorts → ︙（3点メニュー）→「制限付き設定を許可」

---

## 初回設定

1. **アプリを起動**
2. **STEP 1**：「オーバーレイ許可を開く」→ AutoShorts を ON
3. **STEP 2**：「アクセシビリティ設定を開く」→ AutoShorts → ON
4. 各種設定を行う（下記「設定項目」参照）
5. **「オーバーレイボタンを表示」** をタップ
6. YouTube を開くと画面上に **[AUTO]** ボタンが表示される
7. **[AUTO]** をタップ → **[AUTO ON]**（緑）で自動スワイプ開始

---

## 使い方

| 操作 | 動作 |
|---|---|
| **[AUTO] タップ** | AUTO ON（緑）↔ AUTO OFF 切り替え |
| **[AUTO] 長押し** | アプリ設定画面を開く |
| **[AUTO] ドラッグ** | ボタンを画面上の好きな位置に移動 |

---

## 設定項目

### 自動スワイプモード
| モード | 説明 |
|---|---|
| **タイマー** | 設定した秒数（3〜300秒）ごとに自動スワイプ |
| **動画終了を検知** | 動画の赤いプログレスバーを解析して自動スワイプ |

### 動画終了検知モードの検知方式
| 方式 | 説明 |
|---|---|
| **到達％でスワイプ** | バーが設定した％（50〜100%）に達したらスワイプ |
| **ループ検知でスワイプ** | 最後まで再生し、ループ開始（バーのリセット）を検知してスワイプ |

バーが表示されない短い動画は、**バーなし動画のフォールバック時間**（3〜60秒）が経過すると自動スワイプ。

### スワイプ設定
| 項目 | 範囲 | デフォルト |
|---|---|---|
| 方向 | 上 / 下 / 左 / 右 | 上（次の動画） |
| 速度 | 100ms〜800ms | 300ms |
| 距離 | 20%〜80% | 40% |

---

## ⚠️ 注意事項

- 動作は**使用端末・OS バージョン・YouTube アプリのバージョン**に左右されます。
  各種スライダーの数値は実際の動作を見ながら微調整してください。
- **動画終了検知モード**は Android 11（API 30）以上が必要です。Android 10 以下ではタイマーモードを使用してください。
- **アクセシビリティサービス**は端末の再起動後に手動で再有効化が必要な場合があります（Android 仕様）。
- **オーバーレイ許可**は初回のみ手動設定が必要です。
- minSdk = 26（Android 8.0 以上）

├── OverlayService.kt     # フローティングオーバーレイボタン
